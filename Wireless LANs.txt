# Wireless LANs (WLANs) – CCNA Complete Summary Notes

## What is a Wireless LAN (WLAN)?

* A **Wireless LAN (WLAN)** allows devices to connect to a network using **radio waves** instead of cables.
* Uses **Wi-Fi** standards defined by the IEEE.
* Operates primarily in the **2.4 GHz**, **5 GHz**, and **6 GHz** frequency bands.

---

## Main WLAN Components

### Wireless Client

Devices that connect to Wi-Fi:

* Laptops
* Smartphones
* Tablets
* Printers

### Access Point (AP)

* Connects wireless devices to the wired network.
* Acts as a bridge between wireless and wired networks.

### Wireless LAN Controller (WLC)

* Centrally manages multiple APs.
* Common in enterprise networks.

---

## Basic WLAN Topology

```text
Laptop
   \
Phone ---- Access Point ---- Switch ---- Network
   /
Tablet
```

---

## SSID (Service Set Identifier)

* The Wi-Fi network name.
* Identifies a wireless network.

Example:

```text
SSID: Office_WiFi
SSID: Student_WiFi
SSID: Guest_WiFi
```

Users select an SSID to connect.

---

## BSSID

* The MAC address of the AP's radio.
* Uniquely identifies a specific access point.

## Frequency Bands

### 2.4 GHz

Advantages:

* Longer range
* Better wall penetration

Disadvantages:

* More interference
* Fewer channels

---

### 5 GHz

Advantages:

* Faster speeds
* Less interference

Disadvantages:

* Shorter range

---

### 6 GHz

Advantages:

* More available channels
* Less congestion

Disadvantages:

* Shorter range than 2.4 GHz

---

## Channels

### 2.4 GHz Non-Overlapping Channels

```text
1
6
11
```

These are the channels most commonly used to avoid interference.

---

## WLAN Modes

### Ad Hoc Mode

Devices communicate directly.

```text
Laptop ↔ Laptop
```

No AP required.

---

### Infrastructure Mode

Devices communicate through an AP.

```text
Client → AP → Network
```

Most common mode.

---

## CSMA/CA

### Carrier Sense Multiple Access with Collision Avoidance

Used by wireless networks because Wi-Fi devices cannot reliably detect collisions.

Process:

```text
Listen
   ↓
Wait
   ↓
Transmit
```

### Open Authentication

```text
No Security
```

```text
Improved Security
```

### WPA2

```text
AES Encryption
```

---

### WPA3

```text
Strongest Security
```

Newest standard.

## Authentication Methods

### PSK (Pre-Shared Key)

```text
Shared Password
```

Used in homes and small offices.

---

### Enterprise Authentication

Uses:

```text
802.1X
RADIUS
```

Common in enterprise environments.

---

## RADIUS

### Remote Authentication Dial-In User Service

Used for:

* Authentication
* Authorization
* Accounting (AAA)

Default Port:

```text
UDP 1812
```

Accounting:

```text
UDP 1813
```

---

## Roaming

Allows clients to move between APs while maintaining connectivity.

Example:

```text
AP1 → AP2 → AP3
```

User keeps network access while moving.

---

## Lightweight vs Autonomous APs

### Autonomous AP

* Managed individually.
* Configuration done on each AP.

---

### Lightweight AP

* Managed by a WLC.
* Centralized administration.

---

## CAPWAP

### Control and Provisioning of Wireless Access Points

Used between:

```text
Access Point ↔ Wireless LAN Controller
```

Ports:

```text
UDP 5246 → Control
UDP 5247 → Data
```
