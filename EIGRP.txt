# STEP 4

- config eigrp we must enter the AS and should match two routers to become neighbors

## R4

router eigrp 100

- enable on all interfaces not encouraged on real networks

network  0.0.0.0 255.255.255.255

- disable auto summary

do show ip protocols

no auto-summary



- config passive interfaces include loopback

- make int G0/0 not connected gigrp neighbors a passive int 
also do in in loopback interfaces

- router will treat loopback int like any other int and try to send eigrp messages out of it even though its not connected to any device

passive-interface g0/0

passive-interface l0

do show ip protocols
