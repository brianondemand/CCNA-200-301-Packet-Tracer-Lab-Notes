Configuring IPv6  Static Routes


- EUI-64 (Extended Unique Identifier-64) is a method that automatically generates the 64-bit interface ID for an IPv6 address using a device's 48-bit MAC address. 

- It eliminates the need for manual configuration or DHCP by mapping a device's hardware address directly into its network address.

1. Split the MAC Address: The 48-bit MAC address is divided into two 24-bit halves.

2. Inject FFFE: The 16-bit hex value FFFE is inserted exactly between the two halves.

3. Flip the 7th Bit: The 7th bit of the first byte (the Universal/Local bit) is inverted. If it was 0, it becomes 1 (and vice versa). This is done to indicate that the address has been transformed to match the 64-bit standard.
