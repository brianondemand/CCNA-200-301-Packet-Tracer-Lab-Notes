### DNS (CCNA Quick Notes)

- Resolves hostnames to IP addresses.
- Acts like the Internet's phonebook.


www.google.com → 142.250.190.78


**Purpose:**

- Allows users to use names instead of IP addresses.

**DNS Components:**

- DNS Client (Resolver) – Requests name resolution.
- DNS Server – Responds with IP addresses.

**Ports:**

- UDP 53 (most queries)
- TCP 53 (zone transfers, large responses)

**Common Records:**

- A → IPv4 address
- AAAA → IPv6 address
- CNAME → Alias
- MX → Mail server
- NS → Name server

**Cisco Command:**


ip name-server 8.8.8.8


**Exam Reminder:**

* DNS = Name → IP
* DHCP = Assign IP
* NTP = Synchronize Time

### One-Line Memory Trick

**DNS finds addresses, DHCP gives addresses, NTP sets time.**
